Paragráfos

Este é um paragráfo. Um dos elementos mais comuns em documentos HTML. Para criá-lo foi utilizado a tag p.

É possível enfatizar um texto com a tag em, dar ainda mais ênfase com a tag strong ou marcar uma palavra com o elemento mark.

Embora existam os elementos i(itálico) e b(negrito), eles não são mais recomendados, pois o HTML deve focar no conteúdo e semêntica, deixando a estilização para o CSS.

Também podemo demarcar código fonte com o elemento code, ou demarcar entrada de teclado com o elemento kdb.

Podemos forçar uma quebra de linha aqui


como o elemento br.

Podemos usar sub para subscrito, como H2O, ou sup para sobrescrito, como em x 2.
